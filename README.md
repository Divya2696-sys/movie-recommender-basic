# 🎬 Movie Recommendation System

## 📌 Description
This project recommends movies based on similarity using machine learning.

## 🚀 Features
- Input a movie name
- Get top 5 similar movies
- Uses cosine similarity

## 🛠️ Tech Stack
- Python
- Pandas
- Scikit-learn

## ▶️ How to Run
1. Install dependencies:
   pip install pandas scikit-learn
2. Run:
   python main.py

## 📷 Example
Enter movie name: Titanic
Output:
- Avatar
- Avengers
